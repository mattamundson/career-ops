## Playwright MCP extraction function (reference)

The validated function used across all 8 apps this session (and the 3 prior in the overnight handoff):

```javascript
async () => {
  await new Promise(r => setTimeout(r, 3500));
  const buttons = [];
  document.querySelectorAll('button,a').forEach(el => {
    const t = (el.innerText||el.textContent||'').trim();
    if (t.length > 0 && t.length < 30 && /^apply|easy apply/i.test(t)) {
      let atsUrl = null;
      if (el.href) {
        try {
          const u = new URL(el.href);
          if (u.hostname.includes('linkedin.com') && u.pathname.includes('/safety/go'))
            atsUrl = u.searchParams.get('url');
          else if (!u.hostname.includes('linkedin.com')) atsUrl = el.href;
        } catch {}
      }
      buttons.push({ text: t, href: el.href || null, atsUrl });
    }
  });
  const bodyText = document.body.innerText.slice(0, 800);
  const deadSignal = /no longer accepting|no longer available|position.*filled/i.test(bodyText);
  return { title: document.title, buttons, deadSignal, bodyStart: bodyText };
}
```

**Decision matrix from the output:**
- `atsUrl` set → external ATS; use that URL as the Apply URL
- only `Easy Apply` button with LinkedIn `href` → LinkedIn Easy Apply channel; keep LinkedIn URL
- `deadSignal: true` → mark Discarded in tracker + queue
- `buttons: []` + no deadSignal in 300-char body → bump to 800-char body-read and re-check; if still no button, consider DEAD (LinkedIn hides the apply CTA when the listing is closed)
